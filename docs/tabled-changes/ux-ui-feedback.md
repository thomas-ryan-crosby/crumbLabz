# CrumbLabz Website – UX/UI Feedback

Source: External UX/UI review (PDF, March 2026)

---

## Recommended Actions (Priority Order)

### 1. Add More CTAs Between Sections
- Add a CTA after the pain points section and after the process section
- Suggested labels: "Start a Project", "Schedule a Consultation"
- Best practice: CTA every 1–2 sections

### 2. Fix Hover States on Non-Clickable Elements
- Feature cards, solutions blocks, and icon sections have hover effects but don't link anywhere
- Either remove the hover animation or make cards link to solutions/contact page

### 3. Fix Process Section Connector Line
- Thin line runs through steps #2–#4 but not step #1
- Either extend through all steps or remove entirely

### 4. Swap "Disconnected Tools" Icon
- Current icon is unclear
- Replace with something more intuitive (broken chain, unlinked nodes, etc.)

### 5. Fold Banner Bar Into Content or Make It More Distinct
- "Rapid Delivery / Custom Built / No Long Contracts / Ongoing Support" banner is easily skippable
- Either integrate into the pain points section below or make it more visually engaging

---

## Do Later (When Content Exists)

### 6. Add Case Studies / Proof
- Add client case studies, project examples, measurable outcomes
- Format: Client → Problem → Solution → Result
- Wait for real completed projects (Trifecta could be first)

### 7. Strengthen Testimonials
- Add company name, job title, optionally company logos
- Update when real testimonials with detail are available

### 8. Internal Navigation / Cross-Linking
- Link solutions → case studies, process → contact, etc.
- Becomes relevant as more pages are added

---

## Parked (Low Priority / Disagree)

### 9. Brand & Logo Clarity
- Feedback: "CrumbLabz" reads as "Crumbl-abz", similar to Crumbl Cookies
- Suggested bolding "Crumb" or new logo treatment
- **Opinion:** Don't change logo based on one review. Monitor if others raise the same concern.

### 10. More Visuals / Illustrations Overall
- Feedback: site relies heavily on text, add diagrams/illustrations
- **Opinion:** Generic advice. Clean text-based design is fine. Custom illustrations are expensive. Don't add stock visuals just to fill space — that looks worse.
